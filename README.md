# Pokedex App

Using Svelte, Svelte Routing and [PokeAPI](https://pokeapi.co/)

Stilized with TalwindCSS

# How to run

Install all dependecies and then run

To run in local server:
```bash
npm run dev
```

To get static files:
```bash
npm run build
```

Or go to https://thomiwidescreen-svelte-pokedex.netlify.app/
